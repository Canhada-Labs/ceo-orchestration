# Format: <Flip>\t<ADR-ref>\t<token>
#
# One entry per known mismatch. A mismatch is "allowlisted" when
# PLAN-012 carries a debate-revised criterion that the owning ADR has
# not yet been amended to reflect. Per ADR-041 §4, the ADR amendment
# lands in the flip PR itself — so until the flip actually happens,
# the plan intentionally leads the ADR.
#
# Each allowlist entry MUST reference the PLAN-012 debate adjustment
# row (C2/C3/C5/etc.) that introduced the revised criterion.
#
# When a flip PR lands, remove the corresponding rows here AND add the
# amendment to the owning ADR in the same PR. The drift-checker then
# flips to strict for that (Flip, ADR) pair.
#
# Columns are tab-separated. Comments + blank lines ignored.
