# docs-freshness-allowlist — one target per line.

# owner: @<owner>
# quarterly review: every 90 days (next review drops to ROADMAP Sprint 11).
#
# Purpose: permit markdown link targets that the freshness scanner
# (.claude/scripts/check-docs-freshness.py) would otherwise flag as broken.
# Typical candidates:
#   - Future files that Phase 7/8 will create (dated reference, not rot).
#   - Archival targets intentionally moved out of tree.
#   - Third-party paths the repo references but does not own.
#
# Format:
#   - one path per line, exactly as it appears in the [text](path) target
#   - #fragment and ?query are stripped before comparison
#   - `#` starts a comment (ignored), blank lines OK
#

# returned zero broken refs; no entries needed.
